# MuLive — Digital Agency Landing Page
Modern responsive landing page for a digital design and marketing agency, focused on user experience, visual storytelling, and conversion.
🔗 Live Demo: https://grupomulive.com/

## ✨ Overview
MuLive is a conceptual digital agency project designed to showcase how branding, UX/UI design, and frontend development can work together to create engaging and conversion-oriented web experiences.

The project emphasizes:
- Clear visual hierarchy
- Smooth user interactions
- Responsive design across devices
- Strategic content structure for business impact

## 🧠 Key Features
- 📱 Fully responsive layout (mobile-first approach)
- 🎯 Conversion-focused hero section and CTAs
- 🎞️ Interactive carousel and animated UI elements
- 👀 Scroll-based animations using IntersectionObserver
- 📊 Animated counters for visual engagement
- 🧩 Dynamic rendering of services, testimonials, and FAQs
- 📬 Contact form with email delivery and auto-reply via EmailJS
- 🔗 SEO & social sharing optimization with Open Graph

## 🛠️ Tech Stack
| Technology | Purpose |
|---|---|
| HTML5 | Semantic structure |
| CSS3 | Custom UI styling & animations |
| JavaScript (Vanilla) | DOM manipulation & interactivity |
| Bootstrap 5 | Grid system & responsive components |
| Google Fonts (Montserrat) | Typography & branding |
| EmailJS | Client-side email delivery without a backend |

## 📬 Contact Form & Email Service
The contact form is powered by **EmailJS**, enabling email delivery directly from the browser without a backend server.

**How it works:**
1. The user fills out the form (name, email, phone, message)
2. A notification email is sent to the agency with the contact details
3. An auto-reply is automatically sent to the user confirming their message was received

**Files involved:**
- `emailService.js` — handles form submission and EmailJS integration
- Templates configured directly on the EmailJS dashboard
