# Grupo Mulive

Responsive landing page for a digital design and marketing agency, built with HTML, CSS, Vanilla JavaScript and Bootstrap 5.

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic structure |
| CSS3 | Custom styles & animations |
| Vanilla JavaScript | Dynamic content & interactions |
| Bootstrap 5 | Layout & responsive components |
| Google Fonts | Montserrat typography |

## Features
- Responsive navbar with mobile support
- Bootstrap carousel with dynamic slides
- Scroll reveal animations with IntersectionObserver
- Animated counters
- Dynamically rendered services, testimonials and FAQs
- Open Graph meta tags for social sharing

## Deploy
Hosted on [Vercel](https://mulive.vercel.app)
